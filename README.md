🎵 DSP Voice Changer

📌 Overview
DSP Voice Changer is a web-based application developed for the Digital Signal Processing (DSP) course.
It enables users to upload or record audio, apply various voice effects, visualize waveforms and frequency analysis,
and download the processed audio through an interactive and user-friendly interface.

   Features
- Upload or record audio
- Apply multiple voice effects:
  - 👶 Child Voice
  - 👴 Old Man Voice
  - 🏞️ Cave Effect
  - 🔊 Echo
  - 🎧 Additional audio effects
- Waveform visualization
- Frequency spectrum analysis
- Download processed audio
